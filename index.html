Creando elementos con JavaScript

Para agregar un elemento con JS ocuparemos este orden:

  • Obtener la referencia del elemento principal al que se va a agregar, en este caso el elemento principal de este div su id es: "ID-div__containerEjemplo1", entonces creamos una referencia de JS así let divPrincipal = document.getElementById(`ID-div__containerInfoOrden`)
  • Crear nuevo elemento let nuevoDiv = document.createElement(`div`);, hasta aquí ya se creó, pero falta agregarlo a alguna parte del divPrincipal, pero antes de eso vamos a darle un contenido interno al div que acabamos de crear
  • Se agrega el contenido al nuevo elemento nuevoDiv.innerHTML = `< p >Este texto se agregó con JS< / p>`
  • Agregar una clase lo haremos con divNuevo.classList.add(`p__parrafoJavascript`)
